### 📜 Introduction

코드 리뷰어와 리뷰이를 매칭 시켜주는 웹 서비스입니다.   
   
코드 리뷰를 통해 개발자들은 성장하고 새로운 것을 더 알아가기를 원합니다.   
하지만 코드 리뷰를 해주는 사람을 찾기는 쉽지 않은 게 현실입니다.   
코드 리뷰를 진행해 줄 리뷰어와 코드 리뷰 받기를 원하는 리뷰이를 연결해 주면 좋겠다는 생각에 프로젝트를 시작했습니다.   
   
**주요 기능**
- 리뷰어 등록
- 원하는 리뷰어에게 코드 리뷰 요청
- 요청 온 코드 리뷰 수락/거절
- 리뷰에 첨부 된 GitHub PR을 통해 리뷰를 진행 후 수락한 리뷰 완료
   
**서비스 URL** : http://13.124.78.119/
